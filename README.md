# Bodyweight & Bands — Progression Program

A structured full-body workout progression app built with React.
Users can browse exercises across 11 muscle groups, follow 4 training
phases (Foundation → Build → Strength → Advanced), and track
completed exercises per session.

🔗 [Live Demo](https://juliakhoury.github.io/bodyweight-progression/)

---

## About

Built as a learning project while studying React. The base structure
followed a tutorial; I worked through the codebase to understand
component architecture, state management, and GitHub Pages deployment.

## Features

- 4 progressive training phases with weekly scheduling
- 11 muscle groups organized by Push / Pull / Core / Legs
- Per-exercise checkboxes for session tracking
- Session templates and key training principles
- Responsive dark UI

## Tech Stack

- React
- GitHub Pages (deployment)
